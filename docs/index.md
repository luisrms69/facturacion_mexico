# Facturación México

Sistema de facturación electrónica CFDI 4.0 para México, integrado con ERPNext y FacturAPI.io como PAC.

## ¿Qué hace este app?

Extiende ERPNext para emitir, cancelar y gestionar CFDIs (tipos I, E, P) desde Sales Invoice,
con soporte multi-sucursal, addendas, E-Receipts, complementos de pago y CFDI Recibidos.
