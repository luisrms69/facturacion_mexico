# Facturación México

Facturación electrónica CFDI 4.0 para México, integrada con ERPNext y FacturAPI.io como PAC certificado ante el SAT.

Permite emitir, cancelar y gestionar CFDIs tipo I (Ingreso), E (Egreso) y P (Complemento de Pago) desde ERPNext, con soporte multi-sucursal y addendas.
